# pict (pict)

Install pict.

## Example Usage

```json
"features": {
    "ghcr.io/devcontainers-community/features/pict:1": {}
}
```

## Options

| Options | Description | Default |
| --- | --- | --- |
| version | Select or enter a pict version to install (currently only 'latest' is supported, which builds from the main branch). | latest |
